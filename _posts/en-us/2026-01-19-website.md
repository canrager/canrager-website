A list of tools I used to create this website:

- AI coding chatbot (I use Claude Code in the VSCode IDE.)
- Jekyll (Static site generator.)
- multi-language-al-folio (An amazing website template with all the features I need.)
- Strato (Domain host.)
- Netlify (Website backend that connects to GitHub and serves my website.)
- GoatCounter (Privacy-friendly website visitor counter.)
- Loop (Newsletter backend.)

I am grateful that these tools work so well and I am not sponsored by any of these developers.
Only the AI coding chatbot and the domain hosting are paid services.

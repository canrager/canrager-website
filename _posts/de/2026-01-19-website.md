Eine Liste der Tools, die ich zum Erstellen dieser Website verwendet habe:

- AI-Coding-Chatbot (Ich benutze Claude Code in der VSCode IDE.)
- multi-language-al-folio (Eine großartige Website-Vorlage mit allen Funktionen, die ich benötige.)
- Jekyll (Generator für statische Websites.)
- Strato (Domain-Hosting.)
- Netlify (Website-Backend, das sich mit GitHub verbindet und meine Website bereitstellt.)
- GoatCounter (Datenschutzfreundlicher Zähler der Website-Besucher.)
- Loop (Newsletter Backend.)

Ich bin dankbar, dass die Tools so gut funktionieren und werde von diesen Entwicklern nicht unterstützt.
Kostenpflichtig sind ausschließlich der AI-Coding-Chatbot und das Domain-Hosting.
